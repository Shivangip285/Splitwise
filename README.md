Splitwise is an `expense sharing application`.
An expense sharing application is where user can add your expenses and split it among different people. The app keeps balances between people as in who owes how much to whom.
<img width="541" alt="image" src="https://github.com/Shivangip285/Splitwise/assets/106912393/50feaf1e-b44b-4f0f-bd2a-18c1c68acd12">

Application detail

1)User: Each user have a userId, name, email, mobile number.

2)Expense: Could either be EQUAL, EXACT or PERCENT

3)Users can add any amount, select any type of expense and split with any of the available users.

4)The application show expenses for a single user as well as balances for everyone.

5)When asked to show balances, the application show balances of a user with all the users where there is a non-zero balance.
