This Splitwise is an expense sharing application.

An expense sharing application is where user can add your expenses and split it among different people. The app keeps balances between people as in who owes how much to whom.

User: Each user have a userId, name, email, mobile number.
Expense: Could either be EQUAL, EXACT or PERCENT
Users can add any amount, select any type of expense and split with any of the available users.
The application show expenses for a single user as well as balances for everyone.
When asked to show balances, the application show balances of a user with all the users where there is a non-zero balance.
